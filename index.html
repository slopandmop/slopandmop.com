One-click confidential slop

Deploy ollama + openclaw onto a DD agent you own.

Click the button, paste a GitHub token, enter your agent's hostname. This page forks slopandmop/slopandmop into your account, wires up the secrets, and kicks off the deploy workflow. All in your browser — no server, no stored credentials.

Deploy

The public hostname of an agent whose DD_OWNER is a GitHub org you belong to.
Needs repo + workflow scopes. Create one with the right scopes → The token is used directly in your browser; it's written as a repo secret on the new fork and never sent to any server here.
Advanced
Created as <your-login>/<this-name>.
Try qwen2.5:0.5b on CPU-only agents. 

      


      

        

          
What runs

          
podman (static tarball) → bootstrap wrapper → (optional) nvidia driver → ollama serving your model → openclaw gateway in front. All containerised.

        

        

          
Where it lives

          
In an Intel TDX VM sealed against the host. Ingress terminates at a Cloudflare tunnel; openclaw is exposed at openclaw.<agent> automatically.

        

        

          
What you keep

          
A GitHub repo with the workflow. Push changes to redeploy. Delete the branch to trigger the (forthcoming) mop.
